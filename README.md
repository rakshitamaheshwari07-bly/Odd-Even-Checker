# Odd-Even Checker

A simple Python program that checks whether a number is even or odd using if-else conditions.

## Features

* Takes user input
* Checks whether the number is even or odd
* Uses Python if-else statements
* Beginner-friendly project

## Technologies Used

* Python

## Learning Outcomes

* Data Type Conversion
* Modulus Operator (%)
* Conditional Statements (if-else)
